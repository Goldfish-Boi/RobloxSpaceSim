# RobloxSpaceSim
All of the scripts I've made for my roblox space simulator! Initially started during the Summer of Making.

------------------------------Instructions for each UI button------------------------------
UI buttons can be used by inputing a proper value into the textbox and pressing the button's corresponding star icon

Load celestial body: Loads in the specified specified celestial body and it's respective major moons. Current celestial bodies available (Sun, Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus, Neptune, Pluto)

Adjust Time-Scale: Changes the time scaling the simulation operates at. 1 is the default value where one second is equivalent to 1 hr of planetary movement. Numbers higher than 1 will speed up time. Positive numbers less than 1 will slow down time. 0 will stop the simulation and pause time. Negative values will result in the simulation running in reverse.

Camera Lock: Sets the camera's pivot to the celestial body specified, defaults to the world origin

Aim Camera: Locks the camera's direction to always look at the celestial body specified, will additionally display information about the distance (both in in-game studs and real life equivalent kilometers) beteween the camera and the celestial body it is looking at. When combined with Camera lock, you can find the distances between any two celestial bodies at any given time

Size Scale: Changes the size of the in-game system, real life distance equivalents are kept constant. Defualts to 1. Values larger than 1 will make the system bigger. Positive values less than 1 will make the system smaller. Values are clamped from 0.001 and 100.

----------------------------------------------------------------------------------------

Enter the name of any of the eight planets (plus the Sun and Pluto!) in the solar system to view the following:
- Rotation tilt
- Rotational Period
- Accurate orbits of major moons
- Accurate sizes and scales

Use additonal buttons to explore the celestial bodies in each system!
Earth is loaded in by default.

Default time scale: 1s = 1hr
Default size scale: 1stud = 1000km
